Aether Chatbot test page

Please tryout a chat session with the bot in the bottom right corner

For more information see the Chronos Exporter repo.

Conversations you can have with the bot -

  1. To start a conversation, say hi!
  2. You ask questions about any key Aether objects (like enterprise, application, slice, device-group etc.).
  3. To test out the Device Alert workflow, type in a sample alert informational command "[Device Type][Device ID] [Alert Severity] [Time]".
    For example "Phone 765 Critical 12:21".
    Then, you have 2 options - you can either ask the bot questions about the alert (on connectivity, time of connectivity, small cells
    and slices related to the alert etc.), or ask the bot to create a ticket for the issue.

For more information on the workflow, please refer to Rasa Workflow Slides
